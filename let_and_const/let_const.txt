- What is the difference between var and let?

Var can be reassigned and redeclared while let can only be reassigned
A hoisted variable can be accessed with var where a block scope is created with let


- What is the difference between var and const?

Var can be reassigned and redeclared but you cannot do either with const
A hoisted variable can be accessed with var where a block scope is created with const

- What is the difference between let and const?

Let can be reassigned but you cannot reassign or redeclare with const

- What is hoisting?

It explains how the JS compiler works. Variables are lifted to the top of the scope they are declared in.
With var you can access the variable name and its undefined value before its used later on.

Function declarations can be invoked before they are defined in a code base.
